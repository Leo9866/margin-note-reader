# Margin Note Reader Plan

This directory records the product background, original motivation, and planned iterations for Margin Note Reader.

## Documents

- [Project Background](./project-background.md): why this product exists, who it is for, and what problem it is trying to solve.
- [Iteration Plan](./iteration-plan.md): staged roadmap from the current local prototype toward a usable AI-assisted Markdown reading product.

## Current Product Direction

Margin Note Reader is an independent product interface for reading long Markdown documents with structured notes and contextual AI assistance. It is intentionally separate from Hermes Agent. Hermes or another model backend may later provide AI capability, but the product surface should feel like a focused reading desk rather than an agent management dashboard.
